User Guide

Basic Countdown

To start a simple countdown, enter the desired duration in the "Hours", "Minutes", and "Seconds" fields on the main screen and press "Start Countdown".

Customization

  • Background Image: Click "Choose Background Image" to select an image from your device. This background will be saved for your next visit. You can also change it during a live countdown using the camera icon .
  • Final Message: Type a custom message in the "Message to display at the end" box. This message will appear when the countdown finishes.

Advanced Settings (Checkpoints)

For complex timers, you can use Checkpoints. Click "Advanced Settings" to switch to this mode.

  • Adding a Checkpoint: Click "+ Add Checkpoint". A new row will appear.
  • Configuring a Checkpoint: In each row, enter a name for the checkpoint and its specific duration (h, m, s). The total countdown time will be the sum of all checkpoint durations.
  • Removing a Checkpoint: Click the red '×' button on the right of any checkpoint row to delete it.
  • Resetting: Click "Reset All" to clear all checkpoints you've created.
  • Switching Modes: When in Advanced Mode, the timer uses the checkpoint list. To use the simple duration fields again, click "Back to Basic". The app will remember your checkpoints if you switch back to Advanced Mode later.

Warning Time

In Advanced Settings, you can set a "Warning Time". Before each checkpoint ends, the timer will start glowing red and beeping for the duration you set here (default is 10 seconds).

In-Countdown Controls

  • Stop Countdown: Stops the current timer completely and resets all settings.
  • Change Background : Opens a file dialog to select a new background image without stopping the timer.
  • Fullscreen : Toggles fullscreen mode for an immersive experience. This button appears at the bottom-right during a countdown.

Data Persistence

All your settings, including the timer's end time, checkpoints, and customizations, are automatically saved in your browser. If you reload the page, the countdown will resume exactly where it left off.
